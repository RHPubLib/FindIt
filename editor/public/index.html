      How to Use the Rectangle Editor

      Getting Started

      1. Upload a floor plan image or select a saved one from the dropdown
      2. Switch to the Draw tool (click the rectangle icon or press R)
      3. Click and drag on the floor plan to draw a rectangle over a shelf section
      4. Use the Properties panel on the right to label it — pick a collection from Polaris or type manually
      5. Name your project and click Save

      Tools

      VSelect — Click rectangles to select, drag to move, drag handles to resize
      RDraw — Click and drag to draw a new rectangle
      DelDelete the selected rectangle

      Navigating the Floor Plan

      Scroll wheelZoom in/out at cursor position
      + / -Zoom in / out
      0Fit image to view
      Right-click dragPan / move the floor plan around
      Arrow keysPan the floor plan
      Space + dragPan (hold Space, then left-click drag)
      Middle-click dragPan

      Rectangle Properties

      Select a rectangle and use the right panel to set:

      • Collection — Pick from Polaris or type a name
      • Shelf Location — Pick from Polaris shelf locations
      • Call Number Range — Dewey start/end for call number matching
      • Display Label — Text shown to patrons in FindIt
      • Color — Rectangle highlight color (teal default)
      • Position — Fine-tune X, Y, Width, Height as percentages

      Saving & Exporting

      • Save — Saves the project (floor plan + all rectangles) to the server
      • Export JSON — Generates FindIt-compatible config data you can paste into your FindIt configuration
